Include the extended CSS & JavaScript

Before you could use this library, you have to include the official MDL resources. For details, please read the Include the master CSS & JavaScript part of the MDL documentation.

After you included the official MDL resources, include the extended CSS and JavaScript files in each HTML page in your project. I can only provide downloadable version.

Download the minified CSS and Javascript now (9.28kB zipped):

Refer to these files by adding a <link> and a <script> elements into your HTML pages: 
<link rel="stylesheet" href="./material.components.ext.min.css">
<script src="./material.components.ext.min.js"></script>

That's it! You are now ready to add MDL components on your site.
