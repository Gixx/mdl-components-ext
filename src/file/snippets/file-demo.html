MDL-Styled File input element

Demo

NEW - You can reset the file input field in three ways:

  1. Press the form's reset button (input[type=reset] or button[type=reset] ).
  2. From JavaScript reset the form: document.getElementById('myFormElement').reset();
  3. Reset only the file element (also from JavaScript: componentRegister.getComponentById('myFileInputElement').resetInput_();

NOTE: the componentRegister object collects every registered MDL element, and it is able to return the their components' instances, but each component has different methods available.
The resetInput_(); component method is available only for the File component.
