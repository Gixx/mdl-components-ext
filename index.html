About Material Design Light Component Extensions

The MDL

The Material Design Lite (MDL) from Google is a really great thing if you want to create semantic websites with nice looking elements that also fit for mobile devices.

The Idea

Sooner or later you may face the problem that the components provided by the MDL are not covers all your needs. For example you will need a component that works similar to your phone's contact form when you deal with multiple similar data (e.g. phone numbers): add and remove, change type, set value. Or maybe you need a file uploader that has the style of the MDL. And so on.

Legal things

This library provides different components as extensions to the official MDL library. To be able to use this library, you also have to aggree with the official MDL license. Please read the The Material Design Lite license.

You may noticed, this website is using the same design elements (including the Logo as well) as the original Material Design Light Website. This is because all the original MDL resources are linked from their original place, and it's only purpose is to make the visitors feel more comfortable. I hope the guys at The Google don't mind it. Since my library is completely depending on their codes, I thought it's not that bad idea to present them in the same way as they did. I would use the MDL anyway...
